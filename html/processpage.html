Process Page

Planning Stage

During my planning stage, I looked at the Project 2 requirements, and I saw that I need to create 7 total pages using Bootstrap and JavaScript. So, I started to plan out what I wanted to do for each page. I decided to create a personal website that represent my professional online presence. I wanted a photos and videos page that showcases my photography and videography. I also wanted to create a shopping page that sells my (fake) wallpaper packs and design packages. I also wanted to create a blog page that showcases some of my written guides. I also wanted to create a contact page that allows people to contact me. I also wanted to create a resume page that showcases my resume. I also created a blog subpage that showcases one of my written guides.

Figma Prototype

The following image is a screenshot of the prototype of my personal website created using Figma.

When building the Figma prototype, I wanted to create a simple and clean design that is easy to navigate. I started to create frames for the different webpages that were required. Since we are using Bootstrap, my website is designed to be responsive and can be viewed on any device. I also kept in mind the website to be AODA compliant to be accessible to people with disabilities. I also found a Bootstrap UI Kit to help me find appropriate components for the different webpages.

Optimum Store

All these websites were aesthetically pleasing. I liked the font Inter that the Raycast website used. I used the font throughout the website especially the navigation bar. I also took inspiration for the monotone black, white, and grey colors from both Oliur's Store website and the Optimum Store website. I also used some of their products to fill up my shopping page. I took inspiration from Oliur's Portfolio Website because of the website layout.

AODA Compliance

I used the WAVE Evaluation Tool extension to make sure that my website has no major errors and is AODA compliant.
